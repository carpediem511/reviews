## Проект "Карусель отзывов"

### ОСНОВНАЯ ИДЕЯ:

создание интерактивной веб-страницы для отображения и перелистывания отзывов. 

### ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

1. **Отзывы:** Список отзывов, представленных в виде массива объектов, содержащих информацию о заголовке, 
стеке технологий, тексте отзыва, авторе и дате.

2. **Карусель отзывов:** С помощью JavaScript реализован механизм карусели, который позволяет пользователям перелистывать 
отзывы с помощью кнопок "вперед" и "назад". Если пользователь достигает конца списка отзывов, карусель перелистывается снова, 
создавая бесконечный цикл.

3. **Случайный отзыв:** Добавлена кнопка "Случайный отзыв", которая позволяет пользователям просматривать случайный отзыв, 
не следуя порядку.

4. **Структура и стиль:** Отзывы стилизованы с использованием SCSS, обеспечивая приятный дизайн и читабельность.

### ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

- **Node.js:** Необходим для установки и использования сборщика проектов Parcel, а также для автоматизации процессов.

- **Parcel:** Используется для сборки и управления зависимостями проекта.

- **SCSS:** Препроцессор CSS, позволяющий структурировать и улучшить стилизацию веб-страницы.

- **JavaScript:** Для создания динамической карусели отзывов и обработки действий пользователей.

### ОСОБЕННОСТИ ПРОЕКТА:

- Возможность перелистывания отзывов вперед и назад с помощью интуитивных кнопок.

- Возможность просмотра случайного отзыва, что добавляет интересный функциональный элемент.

- Стилизация с использованием SCSS, создающая привлекательный и современный дизайн.

- Практическое использование современных технологий для сборки проекта и улучшения процессов разработки.
